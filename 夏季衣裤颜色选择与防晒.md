# 夏季衣裤颜色选择与防晒

## 核心原则

夏季买衣裤，颜色要同时考虑两件事：

1. 体感是否凉快
2. 是否能有效防晒

简单说：

> 日常通勤优先选浅色，更凉快；长时间户外暴晒时，优先看面料防晒能力，尤其是 UPF 标识。

## 推荐颜色

### 上衣

适合夏天的颜色：

- 白色
- 米白
- 浅灰
- 浅蓝
- 浅卡其
- 浅绿色

这些颜色反光强，吸热少，穿起来更清爽。

### 裤子

更实用的颜色：

- 卡其
- 浅灰
- 浅牛仔蓝
- 军绿
- 藏青
- 深灰

裤子不一定要选特别浅的颜色。浅色裤子容易透、容易脏，卡其、灰色、军绿、藏青会更耐穿。

## 关于太阳和防晒

### 浅色更凉快，但不一定更防晒

浅色衣服会反射更多阳光，所以体感更凉快。但是如果布料很薄、很透，紫外线仍然容易穿过去。

### 深色更吸热，但通常更遮光

黑色、藏青、深灰等深色衣服吸热更明显，暴晒时会更闷。但在同样面料下，深色通常比浅色更能阻挡紫外线。

### 真正防晒优先看面料

买防晒衣或户外衣服时，不要只看颜色，优先看：

- 是否标注 `UPF50+`
- 面料是否密织
- 是否不透光
- 是否轻薄透气
- 是否适合长时间户外

判断方法：

> 把衣服对着光看，如果很透，防晒能力通常不会太好。

## 实用搭配

| 场景 | 推荐搭配 |
|---|---|
| 日常通勤 | 白 T + 卡其裤 |
| 夏季休闲 | 浅蓝衬衫 + 浅灰裤 |
| 稍微正式 | 米白上衣 + 深灰裤 |
| 户外短时间 | 浅色速干衣 + 军绿裤 |
| 长时间暴晒 | UPF50+ 防晒衣 + 帽子 + 墨镜 + 防晒霜 |

## 避坑提醒

- 灰色上衣容易显汗。
- 纯白薄裤子容易透。
- 黑色裤子夏天暴晒会明显发烫。
- 太紧身、不透气的衣服，颜色再浅也会热。
- 只靠衣服颜色防晒不够，长时间户外要配合帽子、墨镜、防晒霜。

## 一句话总结

夏季日常买衣裤，优先选浅蓝、米白、卡其、浅灰、藏青这些实用颜色；如果要长时间晒太阳，颜色不是第一位，面料密度和 `UPF50+` 防晒标识更重要。
